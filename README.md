# Venom Watch 🕷

AI Security Intake Assessment Tool — NIST AI RMF 1.0 aligned

Venom Watch is a 7-phase automated security assessment tool for evaluating 
AI products before organizational deployment. Built for security engineers.

## Features
- 7-phase NIST AI RMF 1.0 aligned assessment engine
- 120+ AI tools in searchable dropdown
- Director-ready risk reports with go/no-go recommendations
- Email OTP authentication
- Security hardened against OWASP Top 10
